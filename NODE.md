# Paperclip

The living source of truth for **Paperclip** — an open-source orchestration and control plane for autonomous AI companies. Paperclip enables running entire companies staffed by AI agents, providing task management, org charts, governance, budget control, and multi-agent coordination.

**Repository:** [paperclipai/paperclip](https://github.com/paperclipai/paperclip)
