# Inbox UX

UX conventions for the inbox view — the triage surface where users review and prioritize incoming work.

## Key Decisions

### Configurable Columns with Show/Hide

Inbox columns are configurable — users can show or hide columns to match their workflow. The Parent Issue column is optional (hidden by default), enabling hierarchy-aware triage when needed without cluttering the default view.

### Tuned Column Widths

The parent issue and time-ago columns use narrowed widths optimized for scan speed. Column sizing prioritizes information density so users can triage quickly without horizontal scrolling.
